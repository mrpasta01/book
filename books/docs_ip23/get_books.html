get_books.php

К оглавлению К сайту

Описание

Метод возвращает массив объектов book. С информацией о книгах.

Принимаемые параметры

Название Тип Описание
count integer, беззнаковое кол-во записей, которые будут возвращены, максимальное значение: 5
offset integer, беззнаковое Смещение, номер записи с которого нужно начать

Возвращаемое значение

Массив объектов типа book

Название Тип Описание
ID integer Уникальный иднетификатор книги
title string Название книги
author integer Автор(ы) книги
year integer Год выпуска книги
Poster string ссыылка на изображение
URL string ссыылка на книгу

Пример запроса

АДРЕС_САЙТА/docs_ip23/get_bookss.php?count=1

Пример ответа

{"books": [{"Author":"Булгаков", "Title":"Мастер и Маргорита", "Year":"1928", "id":1, "POSTER":"https://img4.labirint.ru/rc/d13e94c6220c9accda6a4eb03b9b72aa/363x561q80/books73/725494/cover.jpg?1573626364", "URL":"https://www.labirint.ru/books/725494/"}]}
